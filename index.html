Politique de Sécurité

1. Protection contre les attaques XSS

Nous avons mis en place plusieurs mesures pour protéger notre site contre les attaques XSS (Cross-Site Scripting) :

  • Échappement automatique des données utilisateur
  • Validation stricte des entrées
  • Content Security Policy (CSP) active
  • Protection XSS du navigateur activée

2. Protection contre les attaques CSRF

Pour prévenir les attaques CSRF (Cross-Site Request Forgery), nous utilisons :

  • Tokens CSRF pour toutes les requêtes POST
  • Validation de l'origine des requêtes
  • Cookies sécurisés avec l'attribut SameSite

3. Chiffrement des données

Vos données sensibles sont protégées par :

  • Chiffrement TLS 1.3 pour toutes les communications
  • Hachage sécurisé des mots de passe avec bcrypt
  • Chiffrement des données sensibles en base de données
  • Stockage sécurisé des informations de paiement

4. Connexions HTTPS

Notre site utilise exclusivement HTTPS pour garantir :

  • La confidentialité des données transmises
  • L'authenticité du site
  • L'intégrité des communications
  • La protection contre les attaques de type "man-in-the-middle"

5. Autres mesures de sécurité

  • Protection contre les injections SQL
  • Limitation des tentatives de connexion
  • Surveillance des activités suspectes
  • Mises à jour régulières des systèmes de sécurité
